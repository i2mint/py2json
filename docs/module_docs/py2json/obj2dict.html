py2json.obj2dict.items_with_transformed_keys(d, key_trans=<function <lambda>>, key_cond=<function <lambda>>)[source]¶

A generator of (transformed_key, value) items.

Parameters

  • d – dict (or mapping) to operate on

  • key_trans – Function that is applied to the key

  • key_cond – Function specifying whether to change the key or not

Returns

A generator of (transformed_key, value)

>>> d = {'a': 1, 2: 20}
>>> dict(items_with_transformed_keys(d, lambda x: x * 100, lambda x: isinstance(x, int)))
